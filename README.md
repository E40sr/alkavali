# Alkavali Madslave - Official Website

Welcome to the official website repository for **Alkavali Madslave**, a critically acclaimed singer and composer known for blending atmospheric electronica with raw, emotional vocals and global percussion rhythms.

## Features

- **Responsive Design**: Mobile-first approach with Tailwind CSS
- **Modern UI**: Clean, professional interface with smooth animations
- **Music Section**: Display of albums, EPs, and singles with links to streaming platforms
- **Bio Section**: Artist biography with professional imagery
- **Tour Dates**: Schedule of upcoming performances
- **Contact Form**: Direct messaging and social media links
- **Dark Theme**: Professional dark design with gold accents
